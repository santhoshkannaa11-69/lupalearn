## Flip-Flops

**Flip-flops** are the basic building blocks of sequential logic — circuits that have **memory**.

### SR Flip-Flop

The simplest flip-flop: Set (S) and Reset (R) inputs.

| S | R | Q | Q̅ | State |
|---|---|---|----|-------|
| 0 | 0 | Q | Q̅ | Hold |
| 0 | 1 | 0 | 1 | Reset |
| 1 | 0 | 1 | 0 | Set  |
| 1 | 1 | - | - | Invalid |

### D Flip-Flop

Captures the input D on each clock edge. Used for **registers** — stores 1 bit of data.

### JK Flip-Flop

The "universal" flip-flop — behaves like SR but without the invalid state.

### Applications

- **Registers** — store multiple bits (D flip-flops)
- **Counters** — count clock pulses (JK flip-flops)
- **Memory** — SRAM cells use flip-flops
