## 1. Hook

A multiplexer (MUX) is like a railroad switch — it selects one of several inputs and connects it to the output. The select line determines which input passes through.

## 2. 2-to-1 Multiplexer

```
Input 0 ──╮
          ├──→ AND ──╮
Select ───┤          │
          │          ├──→ OR ──→ Output
Select ───┤          │
          ├──→ AND ──╯
Input 1 ──╯
```

| Select | Output |
|--------|--------|
| 0 | Input 0 |
| 1 | Input 1 |

## 4. Summary

- **Multiplexer** selects one of n inputs based on select lines
- A 2-to-1 MUX uses: 2 AND gates, 1 NOT, 1 OR
- Used inside CPUs for register selection, instruction routing, and data paths
