## 1. Hook

A latch changes whenever its inputs change. But in a CPU, we need circuits that change only at specific moments — on the tick of a clock. This is where **flip-flops** come in.

## 2. Edge Triggering

A flip-flop only reads its inputs on a **clock edge**:

```
Clock:  ▄▄▄▄████▄▄▄▄████▄▄▄▄████
             ↑        ↑
        Rising edge (transitions)
```

## 3. Types of Flip-Flops

| Type | Inputs | Behavior |
|------|--------|----------|
| **SR** | S, R | Set or Reset (same as latch but clocked) |
| **D** | D | Copies D to Q on each clock edge |
| **JK** | J, K | Toggle when both are 1 |
| **T** | T | Toggles output on each clock edge |

### D Flip-Flop (Most Common)

```
D ──→ [DFF] ──→ Q
CLK ──→     ──→ Q̅
```

On each rising clock edge, Q = D. Used for **registers** — stores 1 bit.

## 4. Summary

- **Flip-flops** are edge-triggered (clocked) storage elements
- **D flip-flop** is the most common type
- Used for **registers**, **counters**, and **state machines**
