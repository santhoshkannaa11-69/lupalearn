## 1. Hook

A computer's motherboard is covered with hundreds of components — resistors, capacitors, chips. Each has a specific job, and together they create a functioning system.

## 2. Key Components

| Component | Symbol | Function | Used In |
|-----------|--------|----------|---------|
| **Resistor** | ──www── | Limits current flow | Every circuit |
| **Capacitor** | ──||── | Stores electrical charge | RAM, power filtering |
| **Diode** | ─→|── | Allows current one way | Power regulation |
| **LED** | ─→|/── | Emits light | Indicators, displays |
| **Transistor** | Switch | Amplifies or switches | CPUs, memory |
| **IC** | Chip | Complex circuit on one package | Everything |

## 3. The Integrated Circuit

An **IC (Integrated Circuit)** packages millions (or billions) of transistors into a single chip. The CPU, RAM, and GPU are all ICs.

## 4. Summary

- Basic components (resistors, capacitors) support the operation of **transistors**
- **Transistors** are the active switching elements
- **Integrated circuits** package billions of transistors into chips
- A modern CPU contains over 10 billion transistors
