## 1. Hook

A modern AI model runs on GPUs, which are built from billions of transistors, which are electronic switches controlled by voltage. The entire field of computing — from 0s and 1s to self-driving cars — is built on the same foundation: controlled electron flow through silicon.

## 2. The Journey

```
Electrons (physics)
    ↓
Voltage signals (electricity)
    ↓
Transistors (switches)
    ↓
Logic gates (AND, OR, NOT, NAND)
    ↓
Circuits (adders, multiplexers, flip-flops)
    ↓
CPU components (ALU, control unit, registers)
    ↓
Processor (executes instructions)
    ↓
Software (programs running on hardware)
    ↓
Operating System (manages resources)
    ↓
Applications (browsers, games, AI)
```

## 3. Digital Electronics in AI

Modern AI runs on **GPUs** and **TPUs**, which are essentially massive parallel collections of:

```
Multiply-Accumulate units:
    Input × Weight + Previous Result = New Result

Each MAC is built from:
    AND, OR, XOR gates → adders → multipliers
```

A modern GPU can perform 10¹⁵ MAC operations per second — all using the same transistor-based logic.

## 4. Summary

- Every level of computing builds on the one below it
- **Transistors** → gates → circuits → CPUs → software → AI
- Understanding this chain makes the entire field coherent
- You've now traced computing from electrons to artificial intelligence
