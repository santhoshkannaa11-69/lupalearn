## 1. Hook

The transistor is arguably the most important invention of the 20th century. It's a tiny switch that can be turned on and off by an electrical signal. Billions of them working together create a computer.

## 2. What a Transistor Does

A transistor has three terminals: **Source, Gate, Drain**.

```
Source ──→ [ Gate controls flow ] ──→ Drain
```

| Gate Voltage | Switch State | Current Flows? |
|-------------|--------------|----------------|
| **Low (0)** | OFF | No |
| **High (1)** | ON | Yes |

## 3. Transistor as a Switch

```
Gate=0 (OFF):   Source ──/ /──→ Drain  → 0
Gate=1 (ON):    Source ──────→ Drain  → 1
```

This is the foundation of all digital logic. By connecting transistors in different patterns, we create AND, OR, NOT, and every other logic gate.

## 4. Historical Impact

| Era | Technology | Transistors |
|-----|-----------|-------------|
| **1947** | First transistor (Bell Labs) | 1 |
| **1971** | Intel 4004 CPU | 2,300 |
| **1993** | Pentium | 3.1 million |
| **2024** | Apple M3 Ultra | 184 billion |

## 5. Summary

- A **transistor** is an electronic switch controlled by voltage
- **Gate = 1** → switch ON; **Gate = 0** → switch OFF
- Combining millions/billions of transistors creates logic gates, which create CPUs
