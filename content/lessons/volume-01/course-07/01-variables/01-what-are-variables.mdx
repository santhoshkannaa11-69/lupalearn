## What is a Variable?

A **variable** is a named location in memory that stores a value.

Think of it like a labeled box:

```
┌─────────────────┐
│    age = 25     │
│  [memory cell]  │
│  address: 0xFF  │
└─────────────────┘
```

The **name** (`age`) is how you refer to it. The **value** (`25`) is what it holds. The **memory address** (`0xFF`) is where it lives — you usually don't need to worry about this.

## Why Use Variables?

- Store data for later use
- Make code readable (use `price` instead of remembering `19.99`)
- Change values dynamically as your program runs
- Avoid repeating yourself

## Naming Conventions

Rules for variable names:

- Use letters, numbers, and underscores
- Must start with a letter or underscore
- Case-sensitive (`age` ≠ `Age`)
- Cannot use reserved keywords (`if`, `for`, `while`, etc.)
