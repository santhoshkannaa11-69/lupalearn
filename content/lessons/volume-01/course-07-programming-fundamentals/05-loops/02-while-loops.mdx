## 1. Explanation

A `while` loop runs as long as a condition is true. Use it when you don't know exactly how many iterations you need.

```javascript
while (condition) {
  // code to repeat
}
```

## 2. Code Example

```javascript
// Keep halving until we get below 1
let n = 100;
while (n > 1) {
  console.log(n);
  n = n / 2;
}
// Output: 100, 50, 25, 12.5, 6.25, 3.125, 1.5625

// Reading until a condition is met
let attempts = 0;
let maxAttempts = 5;
while (attempts < maxAttempts) {
  console.log("Attempt:", attempts + 1);
  attempts++;
}
```

## 4. Summary

- While loops run while a condition is true
- Use when you don't know the iteration count
- Always ensure the condition eventually becomes false
- `for` loops are better when you know the count
