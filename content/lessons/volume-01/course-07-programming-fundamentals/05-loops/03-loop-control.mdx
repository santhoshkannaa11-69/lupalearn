## 1. Explanation

Sometimes you need finer control over your loops:

- `break` — **exits** the loop immediately
- `continue` — **skips** to the next iteration

## 2. Code Example

```javascript
// Break: find the first number divisible by 7
for (let i = 1; i <= 100; i++) {
  if (i % 7 === 0) {
    console.log("Found:", i);
    break;  // Stop looking
  }
}

// Continue: skip even numbers
for (let i = 1; i <= 10; i++) {
  if (i % 2 === 0) {
    continue;  // Skip even numbers
  }
  console.log("Odd:", i);
}
// Output: 1, 3, 5, 7, 9
```

## 4. Summary

- `break` exits the loop immediately
- `continue` skips to the next iteration
- Use them sparingly — they can make code harder to follow
