## 1. Explanation

A loop inside another loop is called **nesting**. For each outer iteration, the inner loop runs completely.

```javascript
for (let i = 0; i < 3; i++) {
  for (let j = 0; j < 2; j++) {
    // Runs 6 times (3 × 2)
  }
}
```

## 2. Code Example

```javascript
// Multiplication table
for (let i = 1; i <= 3; i++) {
  for (let j = 1; j <= 3; j++) {
    console.log(i + " × " + j + " = " + (i * j));
  }
}

// Clock: hours and minutes
for (let h = 0; h < 3; h++) {
  for (let m = 0; m < 60; m += 15) {
    console.log(h + ":" + String(m).padStart(2, "0"));
  }
}
```

## 4. Summary

- Nested loops have an outer and inner loop
- Inner loop runs fully for each outer iteration
- Total iterations = outer × inner
- Useful for tables, grids, and patterns
