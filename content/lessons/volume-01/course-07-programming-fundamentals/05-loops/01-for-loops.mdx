## 1. Hook

Print numbers 1 to 100. Without loops, you'd write 100 `console.log` statements. With a loop, you write three lines.

## 2. Core Concept

```javascript
for (initialization; condition; update) {
  // code to repeat
}
```

| Part | What It Does | Example |
|------|-------------|---------|
| `let i = 0` | Runs once at start | Initialize counter |
| `i < 5` | Checked before each run | Stop when false |
| `i++` | Runs after each iteration | Increment |

## 3. Execution Visualization

```
i = 0 → i < 5? true → run body → i++ (now 1)
i = 1 → i < 5? true → run body → i++ (now 2)
i = 2 → i < 5? true → run body → i++ (now 3)
i = 3 → i < 5? true → run body → i++ (now 4)
i = 4 → i < 5? true → run body → i++ (now 5)
i = 5 → i < 5? false → exit loop
```

## 4. Code Walkthrough

```javascript
let sum = 0;
for (let i = 1; i <= 5; i++) {
  sum += i;  // same as: sum = sum + i
}
console.log(sum); // 15 (1+2+3+4+5)
```

## 6. Common Mistakes

| Mistake | Why | Fix |
|---------|-----|-----|
| `for (let i = 0; i <= 5; i++)` | Off by one (runs 6 times) | Use `<` instead of `<=` |
| `for (;;)` | Infinite loop | Always set a condition |

## 7. Summary

- For loops repeat code a **set number** of times
- Three parts: **initialization, condition, update**
- The loop stops when the **condition becomes false**
