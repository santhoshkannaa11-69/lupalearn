## 1. Why This Matters

Imagine printing "Hello" 100 times. Without loops, you'd write 100 `console.log` statements. With a loop, you write ONE.

## 2. Explanation

A `for` loop has three parts:

```javascript
for (initialization; condition; update) {
  // code to repeat
}
```

| Part | What It Does | Example |
|------|-------------|---------|
| `let i = 0` | Runs once at the start | Sets counter to 0 |
| `i < 5` | Checked before each iteration | Stops when i reaches 5 |
| `i++` | Runs after each iteration | Increments counter |

## 3. Code Example

```javascript
// Print numbers 0 through 4
for (let i = 0; i < 5; i++) {
  console.log("Iteration:", i);
}

// Sum numbers 1 to 10
let sum = 0;
for (let i = 1; i <= 10; i++) {
  sum += i;
}
console.log("Sum 1-10:", sum);
```

## 5. Summary

- For loops repeat code a set number of times
- Three parts: initialization, condition, update
- The loop stops when the condition becomes false
