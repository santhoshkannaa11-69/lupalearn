## 1. Hook

Without operators, computers could only store data — not *do* anything with it. Operators are verbs for your data: add, compare, combine, check.

## 2. Core Concept

| Operator | Name | Example | Result |
|----------|------|---------|--------|
| `+` | Addition | `5 + 3` | `8` |
| `-` | Subtraction | `10 - 4` | `6` |
| `*` | Multiplication | `3 * 4` | `12` |
| `/` | Division | `15 / 3` | `5` |
| `%` | Modulus (remainder) | `10 % 3` | `1` |

An **expression** is any combination of values and operators that produces a result.

## 3. Code Walkthrough

```javascript
let a = 10;
let b = 3;
console.log(a + b);  // 13 — addition
console.log(a - b);  // 7  — subtraction
console.log(a * b);  // 30 — multiplication
console.log(a / b);  // 3.333... — division
console.log(a % b);  // 1  — remainder
```

## 5. Summary

- Operators **perform actions** on values
- Arithmetic: `+`, `-`, `*`, `/`, `%`
- An expression combines values and operators into a result
