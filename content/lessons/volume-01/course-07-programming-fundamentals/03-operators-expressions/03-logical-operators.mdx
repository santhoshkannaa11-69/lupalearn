## 1. Explanation

**Logical operators** combine multiple conditions into one true/false result.

| Operator | Name | True When |
|----------|------|-----------|
| `&&` | AND | Both sides are true |
| `\|\|` | OR | At least one side is true |
| `!` | NOT | The value is false (flips it) |

### Truth Tables

```
AND (&&):        OR (||):         NOT (!):
T && T = T       T || T = T       !T = F
T && F = F       T || F = T       !F = T
F && T = F       F || T = T
F && F = F       F || F = F
```

## 2. Code Example

```javascript
let age = 20;
let hasLicense = true;

// AND — both must be true
console.log(age >= 18 && hasLicense);  // true

let isWeekend = false;
let isHoliday = true;

// OR — at least one must be true
console.log(isWeekend || isHoliday);   // true

// NOT — flips the value
console.log(!isWeekend);               // true
```

## 4. Summary

- `&&` requires both conditions to be true
- `||` requires at least one condition to be true
- `!` flips true to false and false to true
- Combine logical operators to build complex conditions
