## 1. Explanation

An **expression** is any piece of code that produces a value. You can build complex expressions by combining smaller ones.

```javascript
// Simple value → expression
42

// Variable → expression
let age = 25;
age

// Operator → expression
age + 5

// Function call → expression
Math.max(10, 20)

// Combined
let total = (price * quantity) + tax;
```

## 2. Code Example

```javascript
// Real-world: Shopping cart total
let price = 29.99;
let quantity = 3;
let taxRate = 0.08;

let subtotal = price * quantity;
let tax = subtotal * taxRate;
let total = subtotal + tax;

console.log("Subtotal: $" + subtotal);
console.log("Tax: $" + tax.toFixed(2));
console.log("Total: $" + total.toFixed(2));
```
