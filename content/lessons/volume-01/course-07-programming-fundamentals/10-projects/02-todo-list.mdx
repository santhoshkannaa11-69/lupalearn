## 1. Hook

Every productivity app — from Apple Reminders to Notion — is essentially a todo list with features built around it. If you can build a todo list, you can understand how most data management apps work.

## 2. Project Brief

Build a todo list manager that can add, list, complete, and remove tasks.

## 3. Requirements

1. `add(todo)` — adds a new todo
2. `list()` — shows all todos with status
3. `complete(index)` — marks a todo as done
4. `remove(index)` — removes a todo
5. `summary()` — shows counts (total, done, pending)

## 4. Approach

```javascript
let todos = [];

function add(todo) {
  todos.push({ text: todo, done: false });
}

function list() {
  todos.forEach((todo, i) => {
    let status = todo.done ? "[✓]" : "[ ]";
    console.log(i + ": " + status + " " + todo.text);
  });
}

function complete(index) {
  if (todos[index]) todos[index].done = true;
}

function remove(index) {
  todos.splice(index, 1);
}
```

## 5. Summary

- Arrays of objects are a **common data pattern**
- Functions **encapsulate** operations
- Real apps manage state — changes in data update the display
