## 1. Hook

This is the final project of Programming Fundamentals. It combines everything you've learned: variables, arrays, objects, functions, loops, conditions, and error handling. If you can build this, you're ready for the next course.

## 2. Capstone Project

Build a weather data management tool that stores, analyzes, and reports on weather readings.

## 3. Requirements

1. **Add reading** — `addReading(city, temp, humidity, condition)`
2. **Get city data** — `getCityData(city)`
3. **Average temperature** — `getAverageTemp(city)`
4. **Hottest day** — `getHottestDay(city)`
5. **City summary** — `getCitySummary(city)`
6. **Compare cities** — `compareCities(c1, c2)`
