## 1. Hook

You've learned variables, loops, conditions, and functions. Now it's time to combine them into a real, playable game.

## 2. Project Brief

Build a number guessing game where the computer picks a random number, and the player tries to guess it.

## 3. Requirements

1. Computer picks a random number between 1 and 100
2. Player enters guesses
3. Tell "Too high", "Too low", or "Correct!"
4. Track the number of attempts
5. Let the player play again

## 5. Approach

```javascript
const secret = Math.floor(Math.random() * 100) + 1;
let attempts = 0;
let guessed = false;

while (!guessed) {
  let guess = parseInt(prompt("Guess (1-100):"));
  attempts++;
  if (guess === secret) {
    console.log(`Correct in ${attempts} attempts!`);
    guessed = true;
  } else if (guess > secret) {
    console.log("Too high!");
  } else {
    console.log("Too low!");
  }
}
```

## 6. Summary

- **Math.random()** generates random numbers
- **While loops** are great when you don't know the iteration count
- Games combine all the concepts you've learned
- Test edge cases: 0, 101, non-numbers
