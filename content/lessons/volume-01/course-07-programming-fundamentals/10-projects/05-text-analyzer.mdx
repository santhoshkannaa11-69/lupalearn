## 1. Project Brief

Build a text analyzer that reads text and produces statistics about it.

## 2. Requirements

1. `wordCount(text)` — number of words
2. `charCount(text)` — number of characters (including/excluding spaces)
3. `sentenceCount(text)` — number of sentences
4. `mostFrequentWords(text, n)` — top N most frequent words
5. `averageWordLength(text)` — average word length
6. `readingTime(text)` — estimated reading time (200 words/min)
