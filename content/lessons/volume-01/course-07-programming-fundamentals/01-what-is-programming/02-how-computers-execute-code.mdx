## 1. Learning Objective

Understand what happens when you press "Run" — how your code becomes action.

## 2. Explanation

Computers don't understand JavaScript, Python, or any high-level language directly. They understand **machine code** — binary instructions (0s and 1s).

Your code goes through a pipeline:

```
Source Code ──→ Compiler/Interpreter ──→ Machine Code ──→ Execution
```

JavaScript uses an **interpreter** (inside your browser or Node.js) that reads your code line by line and executes it immediately.

### The Execution Order

```javascript
console.log("First");   // ← Runs first
console.log("Second");  // ← Runs second
console.log("Third");   // ← Runs third
```

Code runs **top to bottom**, **left to right**.
