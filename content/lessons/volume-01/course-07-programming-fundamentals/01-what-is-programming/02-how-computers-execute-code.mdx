## 1. Hook

When you write `console.log("Hello")` and press Run, the computer doesn't understand that text. It understands only binary — 0s and 1s. Something has to translate your code into something the computer can execute.

## 2. The Translation Pipeline

```
Source Code (JavaScript)
    ↓
Interpreter (translates line by line)
    ↓
Machine Code (binary)
    ↓
Execution
```

JavaScript uses an **interpreter** that reads your code one line at a time and executes it immediately. Other languages use a **compiler** that translates the entire file ahead of time.

## 3. Execution Order

```javascript
console.log("First");   // ← Runs first
console.log("Second");  // ← Runs second
console.log("Third");   // ← Runs third
```

Code runs **top to bottom**, one line at a time.

## 5. Summary

- Computers only understand **binary machine code**
- **Interpreters** translate code line by line (JavaScript, Python)
- **Compilers** translate code ahead of time (C, Rust, Go)
- Code executes **top to bottom**
