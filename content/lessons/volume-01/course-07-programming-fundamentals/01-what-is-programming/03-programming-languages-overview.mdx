## 1. Hook

There are over 700 programming languages. Each was created for a reason — the existing tools didn't solve a particular problem well enough.

## 2. Language Use Cases

| Language | Best For | Used By |
|----------|----------|---------|
| **JavaScript** | Web apps | Every website |
| **Python** | Data science, AI | YouTube, Instagram |
| **Java** | Enterprise, Android | Banking apps |
| **Rust** | Systems, performance | Firefox, Discord |
| **Go** | Cloud services | Docker, Kubernetes |

## 3. Why JavaScript for LupaLearn?

JavaScript runs in your browser — no installation needed. You can write and run code immediately in the Playground without setting up anything.

## 4. Summary

- Different languages exist for **different purposes**
- Core concepts (variables, loops, functions) transfer across languages
- JavaScript is a great first language because **you already have everything needed to run it**
