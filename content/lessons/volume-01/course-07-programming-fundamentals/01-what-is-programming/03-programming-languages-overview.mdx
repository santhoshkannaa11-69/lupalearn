## 1. Learning Objective

Learn what a programming language is and why there are so many to choose from.

## 2. Explanation

A **programming language** is a human-readable way to write instructions that a computer can execute. Different languages are designed for different purposes.

| Language | Best For | Example Users |
|----------|----------|---------------|
| **JavaScript** | Web apps, servers | Websites, Node.js |
| **Python** | Data science, AI, automation | YouTube, Instagram |
| **Java** | Enterprise, Android apps | Banking, Android |
| **Rust** | Systems, performance | Browsers, game engines |
| **Go** | Servers, cloud tools | Docker, Kubernetes |

Most languages share the same concepts: variables, loops, functions, conditions. Once you learn one, picking up another is much easier.
