## 1. Learning Objective

Set up your coding environment — the LupaLearn Playground — and write your first real code.

## 2. The Playground

You're already sitting in a powerful coding environment. The **Playground** has three areas:

| Area | Purpose |
|------|---------|
| **Editor** | Write your code (left panel) |
| **Terminal** | Run commands and see output (bottom) |
| **Preview** | See HTML/CSS results (bottom panel) |

## 4. Try It Yourself

Open the **Playground** from the sidebar and experiment:

1. Write code in the editor
2. Click **Run** to execute
3. See the output in the terminal
4. Try changing the text and running again

## 5. Summary

- The Playground has editor + terminal + preview
- Write code in the editor, run it in the terminal
- You have everything you need to start coding right now
