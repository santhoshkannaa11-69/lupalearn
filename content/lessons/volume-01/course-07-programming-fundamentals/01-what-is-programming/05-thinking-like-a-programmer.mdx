## 1. Learning Objective

Programming is 10% typing code and 90% thinking. Learn how to approach problems like a developer.

## 2. The Four Steps of Problem Solving

| Step | What It Means | Example: Making Tea |
|------|---------------|-------------------|
| **1. Understand** | What exactly is the problem? | "I want a hot cup of tea" |
| **2. Break Down** | What are the small steps? | Boil water, add tea bag, pour water, steep |
| **3. Write Steps** | Turn steps into code comments | `// 1. Boil water` `// 2. Add tea bag` |
| **4. Test** | Does it work? | Is the tea hot? Does it taste right? |

## 3. Code Example

```javascript
// Problem: Calculate the area of a rectangle

// Step 1: Define the inputs
let width = 5;
let height = 10;

// Step 2: Process — calculate
let area = width * height;

// Step 3: Output — show the result
console.log("Area:", area);
```

## 6. Summary

- Programming is mostly thinking, not typing
- Break problems into small steps
- Write steps as comments before coding
- Test your solution to verify it works
