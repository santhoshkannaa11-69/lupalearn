## 1. Hook

Programming is 10% typing and 90% thinking. Most beginners open a code editor and freeze because they try to think in code. Instead, think in steps — then translate those steps into code.

## 2. The Four Steps

| Step | Example: Calculate rectangle area |
|------|-----------------------------------|
| **1. Understand** | "I need to multiply width × height" |
| **2. Break down** | Get width, get height, multiply, show result |
| **3. Write as code** | `let area = width * height` |
| **4. Test** | Does it give the right answer? |

## 3. Code Example

```javascript
// Problem: Calculate the area of a rectangle
let width = 5;
let height = 10;
let area = width * height;
console.log("Area:", area);
```

## 5. Summary

- Think in **steps**, not in code
- **Break problems down** before writing solutions
- Programming is **problem-solving** — the code is just the final step
