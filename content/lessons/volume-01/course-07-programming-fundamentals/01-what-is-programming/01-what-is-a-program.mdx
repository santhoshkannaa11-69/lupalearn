## 1. Learning Objective

By the end of this lesson, you'll understand what a computer program actually is — and realize you already use dozens of them every day.

## 2. Why This Matters

Before you write your first line of code, it helps to know what you're actually creating. A program isn't magic — it's a set of instructions that tells a computer what to do.

## 3. Explanation

A **program** is a sequence of instructions that tells a computer how to perform a specific task. Programs take **input**, **process** it, and produce **output**.

### The IPO Model

```
Input ──→ Process ──→ Output
```

| Step | Description | Example (Calculator) |
|------|-------------|---------------------|
| **Input** | Data the program receives | `5`, `+`, `3` |
| **Process** | The instructions that run | Add the numbers |
| **Output** | The result produced | `8` |

Your web browser, phone apps, video games, and even this learning platform are all programs.

## 4. Code Example

```javascript
// A simple program: input → process → output
let input = "developer";           // Input
let output = "Hello, " + input;   // Process
console.log(output);              // Output
```
