## 1. Hook

Your microwave has a program. Your car has dozens of programs. This website is a program. Every digital experience you've ever had was created by someone writing a sequence of instructions.

## 2. Why This Matters

A program is not magic. It's just a list of instructions — like a recipe. Once you understand that, programming stops being mysterious and becomes learnable.

## 3. Core Concept

A **program** is a sequence of instructions that tells a computer how to perform a task.

### The IPO Model

```
Input → Process → Output
```

| Step | Description | Example |
|------|-------------|---------|
| **Input** | Data the program receives | Numbers to add |
| **Process** | Instructions that run | Addition operation |
| **Output** | Result produced | The sum |

## 4. Code Walkthrough

```javascript
let input = "developer";
let output = "Hello, " + input;
console.log(output);
```

| Line | What It Does |
|------|-------------|
| `let input = "developer"` | Stores the input value |
| `let output = "Hello, " + input` | Processes it by combining text |
| `console.log(output)` | Outputs the result |
