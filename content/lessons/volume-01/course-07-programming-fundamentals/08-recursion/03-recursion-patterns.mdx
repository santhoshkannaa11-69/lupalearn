## 1. Pattern 1: Single Branch (Linear Recursion)

One recursive call per step — like counting down or factorial.

```javascript
function linear(n) {
  if (n <= 0) return;
  console.log(n);
  linear(n - 1);  // One call
}
```

## 2. Pattern 2: Multiple Branches (Tree Recursion)

Multiple recursive calls — like Fibonacci or tree traversal.

```javascript
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);  // Two calls
}
```

## 3. Pattern 3: Divide and Conquer

Split the problem in half, solve each half recursively.

```javascript
function binarySearch(arr, target, left, right) {
  if (left > right) return -1;  // Base case: not found

  let mid = Math.floor((left + right) / 2);

  if (arr[mid] === target) return mid;  // Found!
  if (arr[mid] > target)
    return binarySearch(arr, target, left, mid - 1);
  return binarySearch(arr, target, mid + 1, right);
}
```

## 5. Summary

- Linear recursion: one branch (factorial)
- Tree recursion: multiple branches (fibonacci)
- Divide and conquer: split in half (binary search)
- Each pattern has different use cases
