## 1. Explanation

Sometimes you need to check a condition inside another condition. This is called **nesting**.

```javascript
if (outerCondition) {
  if (innerCondition) {
    // Both conditions are true
  }
}
```

## 2. Code Example

```javascript
let isAuthenticated = true;
let role = "admin";

if (isAuthenticated) {
  console.log("Welcome!");

  if (role === "admin") {
    console.log("You have admin access");
  } else if (role === "editor") {
    console.log("You can edit content");
  } else {
    console.log("You can view content");
  }
} else {
  console.log("Please log in");
}
```

## 5. Summary

- Nest if statements to check multiple levels of conditions
- Keep nesting shallow (max 2-3 levels)
- Consider combining conditions with `&&` when possible
