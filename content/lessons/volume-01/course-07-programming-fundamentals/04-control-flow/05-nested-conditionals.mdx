## 1. Hook

Some decisions depend on other decisions. "If you're a member, check if you have enough purchases for a discount." That's nested conditionals — an if inside an if.

## 2. Code Walkthrough

```javascript
let isLoggedIn = true;
let role = "admin";

if (isLoggedIn) {
  if (role === "admin") {
    console.log("Admin dashboard");
  } else {
    console.log("User dashboard");
  }
} else {
  console.log("Please log in");
}
```

## 3. Execution Visualization

```
isLoggedIn = true → enter outer if
  role = "admin" → enter inner if
    → print "Admin dashboard"
```

## 5. Common Mistakes

- Too many nesting levels (keep to 2-3 max)
- Forgetting that inner `else` belongs to the inner `if`

## 6. Summary

- Nested conditionals = **decisions within decisions**
- Keep nesting **shallow** (2-3 levels max)
- Combine conditions with `&&` when possible
