## 1. Hook

Your phone's alarm checks: "Is it 7 AM?" If yes, ring. Your email filters check: "Is this from my boss?" If yes, mark as important. Every interesting program makes decisions based on conditions.

## 2. Core Concept

```javascript
if (condition) {
  // Runs if condition is true
} else {
  // Runs if condition is false
}
```

## 3. Code Walkthrough

```javascript
let score = 85;
if (score >= 90) {
  console.log("Grade: A");
} else if (score >= 80) {
  console.log("Grade: B");  // ← This runs (85 >= 80)
} else {
  console.log("Grade: C");
}
```

## 4. Execution Visualization

```

score >= 90? → false (skip)
```

## 6. Common Mistakes

| Mistake | Why | Fix |
|---------|-----|-----|
| `if (score = 85)` | `=` assigns, doesn't compare | `if (score === 85)` |
| Missing `{}` | Only first line runs in the block | Always use braces |

## 7. Summary

- `if` runs code when a condition is **true**
- `else` runs when it's **false**
- `else if` checks **additional conditions**
