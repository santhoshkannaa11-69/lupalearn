## 1. Why This Matters

Programs need to make decisions. "If the user is logged in, show the dashboard. Otherwise, show the login page." The `if` statement is how you do this.

## 2. Explanation

```javascript
if (condition) {
  // Runs if condition is true
} else {
  // Runs if condition is false
}
```

The condition is any expression that produces a boolean. If it's `true`, the first block runs. If it's `false`, the `else` block runs.

## 3. Code Example

```javascript
let age = 20;

if (age >= 18) {
  console.log("You can vote!");
} else {
  console.log("Too young to vote.");
}

// Multiple conditions with else if
let score = 85;

if (score >= 90) {
  console.log("Grade: A");
} else if (score >= 80) {
  console.log("Grade: B");
} else if (score >= 70) {
  console.log("Grade: C");
} else {
  console.log("Grade: F");
}
```

## 5. Summary

- `if` runs code when a condition is true
- `else` runs when the condition is false
- `else if` lets you check multiple conditions
- Conditions are boolean expressions
