## 1. Explanation

The **ternary operator** is a shorthand for simple if/else statements. It takes three parts:

```javascript
condition ? valueIfTrue : valueIfFalse
```

## 2. Code Example

```javascript
let age = 20;

// If/else version
let message;
if (age >= 18) {
  message = "Adult";
} else {
  message = "Minor";
}

// Ternary version (same logic)
let message2 = age >= 18 ? "Adult" : "Minor";

console.log(message);   // Adult
console.log(message2);  // Adult
```

## 4. Summary

- Ternary is shorthand for simple if/else
- Format: `condition ? value1 : value2`
- Use for simple assignments, not complex logic
- Don't nest ternaries — it hurts readability
