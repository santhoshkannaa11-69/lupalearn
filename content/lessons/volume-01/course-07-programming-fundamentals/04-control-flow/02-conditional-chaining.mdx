## 1. Explanation

You can combine multiple conditions using logical operators (`&&`, `||`, `!`) inside your if statements.

```javascript
if (age >= 18 && hasLicense) {
  console.log("You can drive");
}
```

## 2. Code Example

```javascript
let age = 22;
let hasID = true;
let isVIP = false;

// Both conditions must be true
if (age >= 21 && hasID) {
  console.log("Welcome to the venue!");
}

// At least one condition must be true
if (isVIP || (age >= 18 && hasID)) {
  console.log("Entry granted");
}
```

## 4. Summary

- Combine conditions with `&&` (AND) and `||` (OR)
- Use parentheses to group conditions
- `else` catches everything that doesn't match
