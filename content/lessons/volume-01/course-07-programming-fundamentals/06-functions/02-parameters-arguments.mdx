## 1. Explanation

**Parameters** are the names listed in the function definition. **Arguments** are the actual values passed when calling the function.

```javascript
// name is a PARAMETER
function greet(name) {
  return "Hi, " + name;
}

// "Alice" is an ARGUMENT
greet("Alice");
```

## 2. Code Example

```javascript
// Default parameters
function greet(name = "Guest") {
  return "Hello, " + name;
}

console.log(greet("Alice"));  // Hello, Alice
console.log(greet());         // Hello, Guest

// Multiple parameters with defaults
function makeBooking(room, nights = 1, breakfast = false) {
  console.log(room + " for " + nights + " nights");
  if (breakfast) console.log("Includes breakfast");
}

makeBooking("Suite", 3, true);
makeBooking("Standard");  // Uses defaults
```

## 4. Summary

- Parameters are the placeholders (definition)
- Arguments are the actual values (call)
- Default parameters make functions flexible
- Order matters — arguments match parameters positionally
