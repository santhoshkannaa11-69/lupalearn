## 1. Challenge: Calculator

Write functions for a simple calculator:

- `add(a, b)` — returns a + b
- `subtract(a, b)` — returns a - b
- `multiply(a, b)` — returns a * b
- `divide(a, b)` — returns a / b (handle division by zero)
- `calculate(a, operator, b)` — uses the functions above

<Terminal mode="challenge" language="javascript" initialCode="// Write your functions here

function add(a, b) { return a + b; }

// Then test:
console.log(calculate(10, &quot;+&quot;, 5));
console.log(calculate(10, &quot;*&quot;, 5));
console.log(calculate(10, &quot;/&quot;, 0));" />
