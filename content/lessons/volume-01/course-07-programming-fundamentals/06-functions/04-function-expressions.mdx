## 1. Explanation

Functions can be stored in variables, passed as arguments, and assigned anywhere a value can go. These are called **function expressions**.

```javascript
// Function declaration
function add(a, b) { return a + b; }

// Function expression (stored in a variable)
const subtract = function(a, b) { return a - b; };

// Arrow function (modern shorthand)
const multiply = (a, b) => a * b;
```

## 2. Code Example

```javascript
// Arrow function — concise syntax
const double = (n) => n * 2;
const square = (n) => n * n;

console.log(double(5));   // 10
console.log(square(5));   // 25

// Functions as arguments
function operate(a, b, operation) {
  return operation(a, b);
}

console.log(operate(10, 5, (x, y) => x + y));  // 15
console.log(operate(10, 5, (x, y) => x * y));  // 50
```

## 4. Summary

- Function expressions store functions in variables
- Arrow functions (`=>`) are a shorter syntax
- Functions can be passed as arguments to other functions
- Arrow functions with single expressions can omit `{}` and `return`
