## 1. Hook

Imagine you're writing a calculator. You need to add numbers in 20 different places. Without functions, you'd write the addition code 20 times. With a function, you write it once and call it by name.

## 2. Core Concept

```javascript
function functionName(parameter) {
  // code to run
  return result;
}
```

| Part | Purpose |
|------|---------|
| `function` | Declaration keyword |
| `name` | How you refer to it |
| `parameters` | Inputs (optional) |
| `return` | Output value (optional) |

## 3. Code Walkthrough

```javascript
function greet(name) {
  return "Hello, " + name + "!";
}
let message = greet("Alice");
console.log(message); // "Hello, Alice!"
```

| Line | What Happens |
|------|-------------|
| `function greet(name)` | Creates a function named `greet` that takes a `name` |
| `return "Hello, " + name` | Combines the greeting with the name |
| `greet("Alice")` | Calls the function with "Alice" as input |

## 4. Execution Visualization

```
greet("Alice") → name = "Alice"
  → return "Hello, Alice!"
  → message = "Hello, Alice!"
  → prints "Hello, Alice!"
```

## 6. Common Mistakes

| Mistake | Why | Fix |
|---------|-----|-----|
| `greet()` without `return` | Returns `undefined` | Add `return value` |
| `greet;` without `()` | Doesn't call the function | Always use `()` to call |

## 7. Summary

- Functions **group code into reusable blocks**
- **Parameters** are inputs, **return** is the output
- Call a function with **name()**
