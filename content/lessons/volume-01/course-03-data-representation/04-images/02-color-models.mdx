## 1. Hook

`#00FF41` — that's LupaLearn green. It means Red=0, Green=255, Blue=65. Every color on your screen is a combination of these three values, and hex is just a compact way to write them.

## 2. Color Models

| Model | Bits Per Pixel | Colors |
|-------|---------------|--------|
| **Grayscale** | 8 | 256 |
| **RGB (True Color)** | 24 | 16,777,216 |
| **RGBA** | 32 | 16.7M + transparency |

### How Hex Colors Work

```
# 00  FF  41
  R   G   B
  0   255 65
```

### Memory Required

```
1,920 × 1,080 at 24-bit RGB = 6.22 MB (uncompressed)
```

## 4. Summary

- **RGB** = 3 channels × 8 bits = 24 bits per pixel
- **Hex colors** are RGB values in base-16
- More bits per pixel = more colors = larger files
