## 1. Hook

A 4K TV has 8,294,400 tiny dots (pixels), each displaying one of 16.7 million colors. Every frame of video on that screen is a grid of numbers — and your eyes interpret it as a continuous image.

## 2. Core Concept

A **pixel** (picture element) is the smallest addressable element in a display.

### Resolution Examples

| Resolution | Pixels | Common Use |
|-----------|--------|------------|
| **720p** | 1,280 × 720 | Old HD |
| **1080p** | 1,920 × 1,080 | Full HD |
| **4K** | 3,840 × 2,160 | Modern TVs |
| **8K** | 7,680 × 4,320 | High-end displays |

### File Size Calculation

```
1920 × 1080 = 2,073,600 pixels
× 24 bits (RGB) = 49,766,400 bits
÷ 8 = 6,220,800 bytes ≈ 6 MB per frame
```

## 3. Summary

- A **pixel** is a single colored dot in a grid
- **Resolution** = width × height (total pixels)
- Image file size = pixels × bits per pixel / 8
