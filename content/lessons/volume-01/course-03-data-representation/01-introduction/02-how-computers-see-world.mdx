## 1. Hook

A photo of a sunset, a paragraph of text, a guitar riff, and a 3D model of a house all look completely different to us. To a computer, they're all the same thing: sequences of bits.

## 2. Core Concept

### How Real-World Data Becomes Binary

```
Real World → Sensor/Sampling → Binary → Storage/Processing
```

| Data Type | Conversion Process | Result |
|-----------|-------------------|--------|
| **Text** | Character → ASCII/Unicode → Binary | 01000001 = 'A' |
| **Image** | Light → Pixel → RGB values → Binary | 24 bits per pixel |
| **Audio** | Sound → Samples → Amplitude values → Binary | 44,100 samples/sec |
| **Video** | Frames of images + audio → Binary | Millions of bits/sec |

## 3. The Universal Language

Binary is the universal language of computing. Once you understand that everything is bits, the differences between data types become just different interpretations of the same underlying representation.

## 4. Summary

- Computers convert all data to **binary** through sampling
- Text, images, audio, and video are all **bits** internally
- The "type" is just how the bits are interpreted
