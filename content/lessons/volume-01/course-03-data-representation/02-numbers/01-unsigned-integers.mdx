## 1. Hook

An 8-bit unsigned integer can store values from 0 to 255. A 16-bit can store 0 to 65,535. A 32-bit can store 0 to 4,294,967,295. The range doubles with every additional bit.

## 2. Core Concept

**Unsigned integers** can only represent non-negative values (0 and positive).

| Bits | Minimum | Maximum | Formula |
|------|---------|---------|---------|
| 8 | 0 | 255 | 2⁸ - 1 |
| 16 | 0 | 65,535 | 2¹⁶ - 1 |
| 32 | 0 | 4,294,967,295 | 2³² - 1 |
| n | 0 | 2ⁿ - 1 | 2ⁿ - 1 |

- **Unsigned integers** = non-negative only
- Range = **0 to 2ⁿ - 1**
- More bits = larger range
