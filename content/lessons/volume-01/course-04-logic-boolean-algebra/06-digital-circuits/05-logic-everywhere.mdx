## 1. Hook

Boolean logic isn't just in computers — it's in traffic lights, washing machines, search engines, and even chess engines. Once you know how to spot it, you'll see logic everywhere.

## 2. Logic in Everyday Technology

| Technology | Boolean Logic Used |
|-----------|-------------------|
| **Traffic light** | AND (all sensors triggered), OR (any pedestrian button) |
| **Smartphone Face ID** | AND (face match + liveness check + recent enough) |
| **ATM** | AND (PIN correct + sufficient balance + not blocked) |
| **Search engine** | AND/OR between search terms |
| **Washing machine** | Complex state machine using Boolean logic |
| **Netflix recommendation** | AND/OR across genre, rating, viewing history |
