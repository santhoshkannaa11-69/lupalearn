## 1. Hook

When a traffic light turns red, it's making a decision: "Is the sensor triggered? If yes, turn red. If no, stay green." That's logic. Every program, every circuit, every AI model makes decisions based on logical rules.

## 2. Why This Matters

Logic is the foundation of computing. Every `if` statement, every database query, every CPU instruction, every search algorithm — all built on the same logical principles developed over 2,000 years.

## 3. Core Concept

**Logic** is the study of correct reasoning — how to reach valid conclusions from given facts.

| Human Logic | Computer Logic |
|-------------|----------------|
| "If it rains, the ground gets wet" | `if (isRaining) { ground.wet = true }` |
| "Either you're a member or you're not" | `isMember === true \|\| isMember === false` |
| "You need a ticket AND an ID" | `hasTicket && hasID` |

## 4. Real-World Logic

| System | Logical Decision |
|--------|-----------------|
| **ATM** | If PIN is correct AND balance > amount → dispense cash |
| **Login** | If username exists AND password matches → grant access |
| **Search** | If keyword matches AND page is indexed → show result |
| **Face ID** | If face scan matches stored data AND liveness check passes → unlock |

## 5. Summary

- **Logic** is the foundation of all computer decision-making
- Computers use **Boolean logic** — true/false, yes/no, 1/0
- Every `if` statement, search filter, and circuit uses logic
