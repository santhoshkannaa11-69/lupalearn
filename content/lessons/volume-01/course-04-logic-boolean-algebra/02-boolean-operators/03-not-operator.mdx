## 1. Hook

NOT is the simplest operator — it takes one input and flips it. `true` becomes `false`. `false` becomes `true`. It's the inverter.

## 2. Core Concept

**NOT** returns the opposite of the input.

| A | NOT A |
|---|-------|
| 0 | 1 |
| 1 | 0 |

### Symbol: `!` (in code) · `¬` (in logic) · `¯` (overbar)

## 3. Real-World Examples

| Scenario | Logic |
|----------|-------|
| **Lock** | NOT unlocked → locked |
| **Login** | NOT logged in → show login page |
| **Validation** | NOT valid → show error message |

## 5. Summary

- **NOT** = invert the input (true → false, false → true)
- Programming symbol: `!`
- Used for negation — "if NOT this, then that"
