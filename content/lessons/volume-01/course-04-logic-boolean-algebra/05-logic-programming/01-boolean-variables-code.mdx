## 1. Hook

Every `if` statement, every `while` loop, every `&&` and `||` — they all depend on Boolean values. Making your Boolean conditions readable is one of the most important programming skills.

## 2. Naming Booleans

```javascript
// Clear names
let isLoggedIn = true;
let hasPermission = false;
let isOver18 = age >= 18;

// Avoid names that don't communicate true/false
let status = true;       // ❌ What does "status" mean?
let isLoggedIn = true;   // ✅ Clear: yes/no question
```

## 3. Common Patterns

```javascript
// Check multiple conditions
let canAccess = isLoggedIn && (isAdmin || hasPermission);

// Negative checks
if (!isLoggedIn) {
  showLoginPage();
}

// Ternary with Boolean
let message = isValid ? "Valid" : "Invalid";
```

## 5. Summary

- Use **clear, descriptive names** for Boolean variables
- Boolean variables should read like **yes/no questions**
- Combine Booleans with `&&`, `||`, and `!` for complex conditions
