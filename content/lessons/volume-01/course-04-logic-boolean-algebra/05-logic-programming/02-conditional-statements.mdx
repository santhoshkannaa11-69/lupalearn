## 1. Hook

An `if` statement is a Boolean expression evaluated at runtime. If the expression is `true`, one block runs. If `false`, another runs. That's all control flow is — Boolean logic directing execution.

## 2. If/Else Patterns

```javascript
// Simple
if (isLoggedIn) {
  showDashboard();
}

// If/else
if (score >= 90) {
  grade = "A";
} else if (score >= 80) {
  grade = "B";
} else {
  grade = "C";
}

// Guard clause
if (!isValid) return null;
```

## 3. Boolean Conditions in Practice

```javascript
// Complex decision: approve a loan
let hasGoodCredit = creditScore > 700;
let hasStableIncome = income > 30000;
let lowDebtRatio = debtToIncome < 0.4;

if (hasGoodCredit && hasStableIncome && lowDebtRatio) {
  approveLoan();
} else {
  denyLoan();
}
```

## 5. Summary

- `if` statements are **Boolean expressions** controlling execution
- Use **guard clauses** to exit early and simplify logic
- Combine conditions with `&&`, `||`, and `!` for complex decisions
