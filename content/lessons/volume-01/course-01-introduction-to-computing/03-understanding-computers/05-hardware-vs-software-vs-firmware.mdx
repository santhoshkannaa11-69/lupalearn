## 1. Hook

Three layers. One computer. Each layer has a different job, different update cycle, and different failure mode. Understanding which is which helps you diagnose problems throughout your career.

## 2. Comparison

| Aspect | Hardware | Software | Firmware |
|--------|----------|----------|----------|
| **Can you touch it?** | Yes | No | No |
| **How to update** | Replace physically | Download & install | Special flash process |
| **Failure mode** | Physical damage | Crash/bug | Device becomes unusable |
| **Developer** | Electrical engineer | Software engineer | Firmware engineer |

## 3. The Three-Layer Model

```
SOFTWARE (Applications + OS)
    ┃ User-downloadable, frequently updated
    ▼
FIRMWARE (BIOS, device controllers)
    ┃ Manufacturer-installed, rarely updated
    ▼
HARDWARE (CPU, RAM, storage)
    ┃ Physical, fixed at manufacture
```

## 4. Summary

- **Hardware** provides the physical capability
- **Software** provides flexibility and user-facing features
- **Firmware** bridges the gap with stable, low-level control
- All three layers must work together for a computer to function
