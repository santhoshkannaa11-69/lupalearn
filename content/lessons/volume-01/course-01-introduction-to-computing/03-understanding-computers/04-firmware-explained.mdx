## Firmware Explained

**Firmware** is software that's permanently stored on a hardware device. It sits between hardware and regular software.

### Firmware vs Software vs Hardware

| | Software | Firmware | Hardware |
|---|----------|----------|----------|
| **Persistence** | Stored on disk | Stored in flash ROM | Physical circuits |
| **Updateable** | Easily | Requires specific process | No |
| **Purpose** | User tasks | Device control | Physical computation |
| **Examples** | Chrome, Word | BIOS, router firmware | CPU, RAM |

### Common Firmware

| Device | Firmware | Purpose |
|--------|----------|---------|
| **Motherboard** | BIOS/UEFI | Boot the computer, configure hardware |
| **SSD** | SSD firmware | Manage flash memory, wear leveling |
| **Router** | Router firmware | Route network traffic |
| **Printer** | Printer firmware | Process print jobs |
| **USB drive** | Controller firmware | Manage flash storage |
| **Smart TV** | TV firmware | Process video input, display |

### BIOS/UEFI

When you turn on a computer, the **BIOS** (or modern **UEFI**) is the first thing that runs. It:

1. Performs a power-on self-test (POST)
2. Detects hardware components
3. Finds a bootable device
4. Loads the operating system

Firmware updates can fix bugs and add features — that's why manufacturers occasionally release "BIOS updates."
