## What Makes a Computer?

Every computer, from a smartphone to a supercomputer, has four core functions:

### The Four Functions

```
1. Input    → Accept data
2. Process  → Manipulate data
3. Output   → Display results
4. Store    → Save for later
```

### Essential Components

| Component | Function | Examples |
|-----------|----------|----------|
| **Input devices** | Get data in | Keyboard, mouse, touchscreen, microphone |
| **Processor (CPU)** | Execute instructions | Intel Core, Apple M-series, Qualcomm |
| **Memory (RAM)** | Temporary storage | DDR5, LPDDR5 |
| **Storage** | Permanent storage | SSD, HDD |
| **Output devices** | Show results | Monitor, speakers, printer |

### Types of Computers

| Type | Use | Example |
|------|-----|---------|
| Desktop | Home/office work | Dell, HP |
| Laptop | Portable computing | MacBook, ThinkPad |
| Smartphone | Mobile computing | iPhone, Galaxy |
| Server | Serving data | AWS, Google Cloud |
| Embedded | Specific tasks | Car computers, routers |
| Supercomputer | Scientific computing | Fugaku, Summit |

All of these share the same fundamental architecture — just at different scales.
