## 1. Hook

A smartphone, a laptop, a car's navigation system, and a space station's guidance computer all have one thing in common: they share the same fundamental architecture. Despite looking completely different, every computer does the same four things.

## 2. The Four Core Functions

```
         ┌──────────┐
Input ──→│ COMPUTER │──→ Output
         │          │
         │  Store   │
         └──────────┘
```

| Function | What It Means | Examples |
|----------|--------------|----------|
| **Input** | Accept data | Keyboard, touch, sensor |
| **Process** | Manipulate data | CPU executing instructions |
| **Output** | Return results | Display, sound, signal |
| **Store** | Save for later | Memory, storage |

## 3. Essential Components

| Component | Function | Analogy |
|-----------|----------|---------|
| **CPU** | Executes instructions | The brain |
| **RAM** | Temporary workspace | The desk |
| **Storage** | Permanent data | The filing cabinet |
| **Motherboard** | Connects everything | The nervous system |
| **Power supply** | Provides energy | The heart |

## 4. Summary

- Every computer performs four functions: **input, process, output, store**
- The physical components (CPU, RAM, storage, motherboard) work together to perform these functions
- From smartphones to supercomputers — same architecture, different scale
