## 1. Hook

If you opened your computer right now, you'd see a green board with chips, slots, and wires. It might look like random electronic parts — but every component has a specific job.

## 2. Internal Components

| Component | Purpose | Key Spec |
|-----------|---------|----------|
| **Motherboard** | Connects all parts | Form factor (ATX, mATX) |
| **CPU** | Executes instructions | GHz, cores, threads |
| **RAM** | Active memory | GB, speed (MHz) |
| **SSD** | Permanent storage | GB, read/write speed |
| **GPU** | Graphics processing | VRAM, CUDA cores |
| **PSU** | Power supply | Watts, efficiency |

## 3. How They Work Together

```
Power button pressed → PSU provides power
    → Motherboard starts → CPU loads BIOS
    → BIOS detects hardware → Finds boot drive
    → OS loads into RAM → CPU executes OS
    → You see the login screen
```

## 4. Summary

- Hardware components each have a **specific role**
- The **motherboard** connects everything
- The **CPU** executes instructions from **RAM**
- **Storage** holds data permanently when the power is off

If hardware is the body, **software** is the mind. The next lesson explores software — the instructions that tell hardware what to do.
