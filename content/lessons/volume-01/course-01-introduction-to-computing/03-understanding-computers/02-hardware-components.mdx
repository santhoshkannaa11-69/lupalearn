## Hardware Components

**Hardware** is the physical parts of a computer that you can touch.

### Internal Components

| Component | Purpose | How It Works |
|-----------|---------|--------------|
| **Motherboard** | Connects everything | Circuit board with buses |
| **CPU** | Executes instructions | Billions of transistors switching on/off |
| **RAM** | Active memory | Fast but loses data when power is off |
| **SSD/HDD** | Permanent storage | Magnetic or flash storage |
| **GPU** | Renders graphics | Parallel processing cores |
| **PSU** | Provides power | Converts AC to DC power |
| **Cooling** | Prevents overheating | Fans, heat sinks, liquid cooling |

### Peripheral Devices

| Category | Examples |
|----------|----------|
| **Input** | Keyboard, mouse, touchpad, microphone, webcam, scanner |
| **Output** | Monitor, speakers, headphones, printer |
| **Storage** | USB drive, external hard drive, SD card |
| **Network** | Wi-Fi adapter, ethernet card, modem |

### The Motherboard

Everything connects to the motherboard. It's the central hub that allows components to communicate. The **bus** system on the motherboard moves data between the CPU, RAM, and other components at billions of transfers per second.
