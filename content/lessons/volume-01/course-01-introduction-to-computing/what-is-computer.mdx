## What Is a Computer?

A **computer** is an electronic device that processes data according to instructions. It takes **input**, performs **processing**, and produces **output**.

### The IPO Model

```
Input ──→ Process ──→ Output
```

Every computer follows this model. A calculator takes numbers (input), adds them (process), and shows the result (output).

### Types of Computers

- **Desktop** — Personal computers for everyday use
- **Laptop** — Portable personal computers
- **Server** — Computers that provide services to other computers
- **Smartphone** — Handheld computers with communication capabilities
- **Embedded** — Computers built into other devices (cars, appliances)

All computers share the same fundamental architecture: CPU, memory, storage, and input/output systems.
