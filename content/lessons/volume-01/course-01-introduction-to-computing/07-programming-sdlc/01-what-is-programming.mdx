## What Is Programming?

**Programming** is the process of creating instructions that a computer can execute to perform a specific task.

### What Programmers Do

- **Write code** — Create instructions in a programming language
- **Design solutions** — Figure out how to solve problems
- **Test and debug** — Find and fix errors
- **Maintain code** — Update and improve existing software
- **Collaborate** — Work with other developers

### Programming as Problem-Solving

Programming is not about memorizing syntax. It's about:

1. **Understanding** the problem
2. **Breaking it down** into smaller steps
3. **Designing** a solution
4. **Implementing** the solution in code
5. **Testing** that it works
6. **Iterating** to improve

### The Input-Output View

```
Input (data) → Program (instructions) → Output (result)
```

Your program takes data, processes it according to your instructions, and produces a result.
