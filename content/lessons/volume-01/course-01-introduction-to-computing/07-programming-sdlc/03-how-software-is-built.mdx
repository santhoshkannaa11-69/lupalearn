## 1. Hook

A modern web application isn't built by one person typing code in a dark room. It's built by a team of people with different specialties, using a set of tools that have been refined over decades.

## 2. The Team

| Role | What They Do |
|------|-------------|
| **Product Manager** | Decides what to build |
| **Designer** | Designs the user experience |
| **Developer** | Writes the code |
| **QA Tester** | Finds bugs before users do |
| **DevOps** | Deploys and maintains |

## 3. Essential Tools

| Tool | Purpose | Examples |
|------|---------|----------|
| **Version Control** | Track code changes | Git, GitHub |
| **Editor/IDE** | Write code | VS Code |
| **Package Manager** | Manage libraries | npm, pip |
| **CI/CD** | Auto test/deploy | GitHub Actions |

## 4. Summary

- Software is built by **teams**, not individuals
- Modern development relies on **specialized tools**
- **Version control** (Git) is the most important tool to learn
