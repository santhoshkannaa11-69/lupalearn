## 1. Hook

There are hundreds of programming languages. Each one is a tool designed for specific kinds of problems. JavaScript for the web. Python for data science. Rust for systems programming. C++ for games.

## 2. Levels of Languages

```
High-level (human readable)  Python, JavaScript, Java
        ↓
Mid-level                    C, C++, Rust
        ↓
Low-level                    Assembly
        ↓
Machine code (binary)        10110010
```

## 3. Compiled vs Interpreted

| Approach | How It Works | Examples |
|----------|-------------|----------|
| **Compiled** | Source → Machine code (ahead of time) | C, Rust, Go |
| **Interpreted** | Source → Run (line by line) | Python, JavaScript |
| **JIT** | Source → Intermediate → Machine (at runtime) | Java, C# |

## 4. Summary

- Programming languages are **tools for different jobs**
- **Higher-level** languages are easier to read but offer less control
- **Compiled** languages run faster; **interpreted** languages are more flexible
