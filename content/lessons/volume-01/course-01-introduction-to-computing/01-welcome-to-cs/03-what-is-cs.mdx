## 1. Hook

When people hear "Computer Science," they usually think "programming." But here's the truth: many of the most influential computer scientists rarely write production code.

AI researchers design algorithms. Security experts find vulnerabilities. Network architects design systems that serve billions. Some computer scientists never ship a single app — yet their work changes the world.

So what is CS really about?

## 2. Why This Matters

Understanding the full breadth of CS helps you:

- **Choose** a specialization that excites you
- **See** how different branches connect
- **Avoid** the trap of thinking CS equals programming

## 4. Core Concept

**Computer Science** is the study of **computation** — what can be computed, how to compute it efficiently, and how to build systems that perform computation.

### The Branches of CS

| Branch | Core Question | Example |
|--------|--------------|---------|
| **Theory** | What can be computed? | Algorithms, complexity, P vs NP |
| **Systems** | How do we build reliable software? | Operating systems, databases, compilers |
| **AI/ML** | How do we make computers intelligent? | Neural networks, NLP, robotics |
| **Networks** | How do computers communicate? | Internet protocols, routing, security |
| **Security** | How do we protect data? | Cryptography, authentication, threats |
| **Graphics** | How do we render images? | 3D rendering, game engines, VR |
| **HCI** | How do humans interact with computers? | UI/UX design, accessibility |

## 5. Mental Model — The Library Analogy

```
Computer Science is like a library:

Theory          → The index — what's possible
Systems         → The building — infrastructure
AI              → A librarian who learns your preferences
Networks        → The inter-library loan system
Security        → Locks, alarms, and access control
Graphics        → The art and photography section
HCI             → Signs and navigation
```

## 6. Real-World Applications by Industry

| Industry | CS Application | Branch Used |
|----------|---------------|-------------|
| **Medicine** | AI detects tumors in X-rays | AI/ML, Graphics |
| **Finance** | Fraud detection in real-time | Systems, Security |
| **Entertainment** | Streaming Netflix to millions | Networks, Systems |
| **Transportation** | Self-driving cars | AI/ML, HCI |
| **Communication** | Video calls across continents | Networks, Theory |
| **Science** | Protein folding prediction | AI/ML, Theory |

## 7. Common Misconception

> "Computer Science is just programming."

Programming is a *tool* of CS, not the field itself. Like how a telescope is a tool of astronomy, not astronomy itself.

## 9. Summary

- CS is the **science of computation**, not just programming
- Major branches: **Theory, Systems, AI, Networks, Security, Graphics, HCI**
- Every industry relies on CS in different ways
- Your journey will expose you to all these branches
