## What Is Computing?

**Computing** is any goal-oriented activity that uses computers to process data. It includes designing hardware, writing software, and processing information.

### Computing in Everyday Life

You interact with computing systems dozens of times daily:

| Activity | Computing Involved |
|----------|-------------------|
| Sending a message | Data transmission, encryption, storage |
| Watching a video | Streaming, compression, rendering |
| Ordering food | Database queries, payment processing |
| Using a map | GPS, routing algorithms, real-time data |
| Checking email | Mail servers, spam filters, search |

### The Core of Computing: Data Processing

All computing follows the same pattern:

```
Input Data → Process → Output Data → Store
```

### Automation

Computers excel at automating repetitive tasks. Tasks that would take a human hours can be done in milliseconds.
