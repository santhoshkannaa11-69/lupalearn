## 1. Hook

You press a button on your microwave, and 90 seconds later your food is hot. You tap your phone and a car arrives at your location. You type a question and an AI answers in seconds.

Each of these is a computing system at work. But what's actually happening inside?

## 2. Why This Matters

Every piece of technology you interact with follows the same fundamental pattern. Once you understand this pattern, you'll see it everywhere — and you'll understand what computers actually *do* rather than treating them as magic boxes.

## 3. Learning Objectives

- **Define** computing in terms of data processing
- **Trace** the input-process-output cycle through everyday systems
- **Identify** computing devices you didn't realize were computers

## 4. Prerequisite Recap

In the last lesson, you learned that Computer Science is problem-solving, not just programming. Now we'll look at what computing actually *is* at a fundamental level.

## 5. Core Concept

**Computing** is the process of taking data, performing operations on it, and producing a result.

### The Universal Pattern

```
Input → Process → Output
```

Every computing device follows this cycle.

| Device | Input | Process | Output |
|--------|-------|---------|--------|
| **Microwave** | Button press, time | Generate microwaves, start timer | Food heated, beep |
| **Phone** | Touch, voice | Run app, fetch data, render | Display, sound |
| **Car** | Gas pedal, steering | Engine control, navigation | Motion, direction |
| **AI** | Question text | Pattern matching, prediction | Answer text |

## 6. Mental Model — The Factory Analogy

Think of a computer like a factory assembly line:

```
Raw materials (data) arrive at the loading dock
                ↓
Factory workers (CPU) process each piece
                ↓
Finished products (output) leave the shipping area
                ↓
Some products go to storage (memory) for later
```

## 7. Computing in Daily Life

| Activity | Computing Involved | You Don't See |
|----------|-------------------|---------------|
| **Sending a message** | Encode, transmit, route, deliver | 4+ protocols, multiple servers |
| **Watching a video** | Decode, buffer, render frames | 24-60 calculations per second |
| **Ordering food** | Query database, process payment, notify restaurant | 10+ services coordinating |
| **Using GPS** | Receive satellite signals, calculate position, draw map | Triangulation from 4+ satellites |

## 8. Execution Visualization

```
WHAT HAPPENS WHEN YOU SEND "HELLO":

You type "Hello"
       ↓

       ↓
Data sent as radio waves to cell tower
       ↓
Tower routes through fiber optic cables
       ↓
Server receives, stores in database, sends notification
       ↓
Recipient's phone decodes → displays "Hello"
```

## 11. Summary

- **Computing** = input → process → output
- Every computing device follows this pattern — from microwaves to AI
- The same data can be processed in many different ways
- Understanding this pattern helps you think like a programmer
