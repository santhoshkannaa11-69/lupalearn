## Careers in Computer Science

Computer Science offers diverse career paths. Here are the most common roles.

### Software Engineer

Builds and maintains software applications.

- **Skills**: Programming, system design, testing
- **Languages**: JavaScript, Python, Java, C++
- **Average salary**: $110k-$160k
- **Paths**: Frontend, backend, full-stack, mobile

### Web Developer

Specializes in websites and web applications.

- **Skills**: HTML, CSS, JavaScript, frameworks
- **Frameworks**: React, Next.js, Vue, Angular
- **Average salary**: $80k-$140k
- **Paths**: Frontend, backend, full-stack

### Data Scientist

Extracts insights from data using statistics and machine learning.

- **Skills**: Statistics, machine learning, data visualization
- **Languages**: Python, R, SQL
- **Average salary**: $120k-$170k

### AI/ML Engineer

Builds intelligent systems that learn from data.

- **Skills**: Deep learning, NLP, computer vision
- **Frameworks**: PyTorch, TensorFlow, LangChain
- **Average salary**: $130k-$190k

### DevOps Engineer

Manages infrastructure, deployment, and reliability.

- **Skills**: Cloud, CI/CD, containers, monitoring
- **Tools**: Docker, Kubernetes, AWS, Terraform
- **Average salary**: $120k-$170k

### Cybersecurity Analyst

Protects systems and data from threats.

- **Skills**: Network security, cryptography, ethical hacking
- **Certifications**: CISSP, CEH, Security+
- **Average salary**: $100k-$150k

### Other Paths

| Role | Focus |
|------|-------|
| **Game Developer** | Game engines, graphics, physics |
| **Mobile Developer** | iOS (Swift), Android (Kotlin), cross-platform |
| **Cloud Engineer** | Cloud infrastructure, scaling |
| **QA Engineer** | Testing, automation, quality |
| **Technical Writer** | Documentation, tutorials |
| **Product Manager** | Strategy, roadmaps, user research |
| **Developer Advocate** | Community, education, content |

### How to Choose

Ask yourself:
- Do I enjoy building visual things? → **Frontend/Web**
- Do I enjoy logic and data? → **Backend/Data**
- Do I enjoy AI and intelligence? → **AI/ML**
- Do I enjoy security and puzzles? → **Security**
- Do I enjoy systems and infrastructure? → **DevOps/Cloud**

Your first job doesn't have to be your last. Many engineers switch roles as they grow.
