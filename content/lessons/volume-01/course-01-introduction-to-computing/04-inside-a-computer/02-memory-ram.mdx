## Memory (RAM)

**RAM** (Random Access Memory) is your computer's short-term memory. It holds data that the CPU needs to access quickly.

### How RAM Works

When you open a program:
1. The program's code loads from the **SSD/HDD** (slow, permanent)
2. It's copied into **RAM** (fast, temporary)
3. The **CPU** reads instructions from RAM
4. When you close the program, RAM is freed

### RAM vs Storage

| Aspect | RAM | SSD/HDD |
|--------|-----|---------|
| **Speed** | ~50 GB/s | ~5 GB/s |
| **Capacity** | 8-64 GB typically | 256 GB-4 TB typically |
| **Persistence** | Data lost when power is off | Data persists |
| **Cost per GB** | Higher | Lower |
| **Purpose** | Active work area | Permanent library |

### Types of RAM

| Type | Used In | Speed |
|------|---------|-------|
| **DDR4** | Older computers | Up to 3200 MT/s |
| **DDR5** | New computers | Up to 6400 MT/s |
| **LPDDR5** | Laptops, phones | Power-efficient |

### Why RAM Matters

More RAM allows you to run more programs simultaneously without slowdown. 8 GB is minimum for most users, 16 GB is recommended, and 32+ GB for power users.

When RAM is full, the OS uses **swap space** (a portion of the SSD as slow RAM), which dramatically slows down your computer.
