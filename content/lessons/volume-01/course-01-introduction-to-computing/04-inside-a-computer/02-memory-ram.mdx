## 1. Hook

When your computer slows down with too many tabs open, you're not running out of "storage" — you're running out of RAM. These are two very different things, and confusing them is one of the most common tech misconceptions.

## 2. RAM vs Storage

```
        CPU
         ↑
    Very fast
         ↑
       ┌───┐
       │RAM│ ← Temporary, fast, limited (8-64 GB)
       └───┘
         ↑
    Fast enough
         ↑
       ┌───┐
       │SSD│ ← Permanent, slower, larger (256 GB - 4 TB)
       └───┘
```

## 3. How RAM Works

| Aspect | RAM | SSD |
|--------|-----|-----|
| **Speed** | ~50 GB/s | ~5 GB/s |
| **Persistence** | Lost on power off | Survives reboots |
| **Purpose** | Active workspace | Permanent storage |
| **Upgrade** | Easy (desktop) | Easy |

## 4. Summary

- RAM is **temporary workspace** — fast, small, volatile
- Storage is **permanent library** — slower, larger, persistent
- When RAM is full, your computer uses **swap space** on the SSD (very slow)
