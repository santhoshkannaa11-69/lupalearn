## CPU — The Brain of the Computer

The **Central Processing Unit (CPU)** is the most important component in a computer. It executes instructions that make everything happen.

### The Fetch-Decode-Execute Cycle

```
Fetch → Decode → Execute → (repeat)
```

1. **Fetch** — Get the next instruction from RAM
2. **Decode** — Translate the instruction into commands
3. **Execute** — Perform the operation (math, move data, etc.)

This cycle happens billions of times per second.

### CPU Components

| Component | Function |
|-----------|----------|
| **Control Unit (CU)** | Manages instruction flow |
| **Arithmetic Logic Unit (ALU)** | Performs calculations (+, -, ×, ÷, AND, OR) |
| **Registers** | Ultra-fast internal memory |
| **Cache** | Small, fast memory (L1, L2, L3) |

### CPU Speed

Modern CPUs operate at 2-5 GHz (gigahertz), meaning they cycle 2-5 billion times per second. But speed alone doesn't tell the whole story — architecture and core count matter too.

### Cores and Threads

| Term | Meaning |
|------|---------|
| **Single-core** | One processing unit |
| **Multi-core** | Multiple processing units (e.g., 8-core) |
| **Hyper-threading** | Each core handles 2 threads |
| **Thread** | A sequence of instructions |

A 16-core CPU can handle 32 threads simultaneously — dramatically faster than a single-core CPU at the same speed.
