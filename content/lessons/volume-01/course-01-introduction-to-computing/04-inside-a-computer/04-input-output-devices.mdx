## 1. Hook

When you move your mouse, a signal travels from the mouse to the computer, through the OS and driver, to the application, which then sends a signal back to the monitor — all in under 10 milliseconds.

## 2. Input vs Output

| Input Devices | Output Devices |
|---------------|----------------|
| Keyboard | Monitor |
| Mouse | Speakers |
| Microphone | Headphones |
| Camera | Printer |
| Scanner | Haptic feedback |

## 3. How Devices Communicate

```
Input: Device → Driver → OS → Application
Output: Application → OS → Driver → Device
```

Every device communicates through a **driver** — a translator between the device's native language and the OS's language.

## 4. Summary

- **Input** devices send data to the computer
- **Output** devices receive data from the computer
- **Drivers** bridge the communication gap
- **USB** is the most common connection standard
