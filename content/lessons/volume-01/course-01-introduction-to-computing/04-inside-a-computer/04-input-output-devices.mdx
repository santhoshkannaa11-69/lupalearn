## Input & Output Devices

**Input devices** send data to the computer. **Output devices** receive data from the computer.

### Input Devices

| Device | What It Captures | How It Works |
|--------|-----------------|--------------|
| **Keyboard** | Text input | Key presses → electrical signals → key codes |
| **Mouse** | Position and clicks | Optical sensor tracks movement |
| **Touchscreen** | Touch position | Capacitive grid detects finger location |
| **Microphone** | Audio | Sound waves → electrical signals → digital samples |
| **Camera** | Visual images | Light → sensor → pixel data |
| **Scanner** | Documents | Reflected light → digital image |

### Output Devices

| Device | What It Produces | How It Works |
|--------|-----------------|--------------|
| **Monitor** | Visual display | Pixels lit by LCD/LED/OLED |
| **Speakers** | Sound | Digital signals → analog waves → sound |
| **Headphones** | Private audio | Same as speakers, personal |
| **Printer** | Physical documents | Ink/toner applied to paper |

### Universal Serial Bus (USB)

Most devices connect via **USB** (Universal Serial Bus). USB is a standard that allows any device to communicate with the computer using a common protocol.

| USB Version | Speed | Year |
|-------------|-------|------|
| USB 2.0 | 480 Mbps | 2000 |
| USB 3.0 | 5 Gbps | 2008 |
| USB 3.2 | 20 Gbps | 2017 |
| USB4 | 40 Gbps | 2019 |

### Device Drivers

Every device needs a **driver** — software that tells the operating system how to communicate with that specific hardware.
