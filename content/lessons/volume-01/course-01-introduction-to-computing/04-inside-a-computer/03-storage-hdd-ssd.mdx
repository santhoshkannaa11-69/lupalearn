## 1. Hook

The difference between a 10-second boot time and a 30-second boot time is usually the same thing. The difference between waiting 30 seconds for a file to copy and 30 minutes? Also the same thing.

## 2. HDD vs SSD

| Factor | HDD | SSD |
|--------|-----|-----|
| **Technology** | Spinning magnetic platters | Flash memory chips |
| **Speed** | ~150 MB/s | ~5,000 MB/s (NVMe) |
| **Durability** | Fragile (moving parts) | Tough (no moving parts) |
| **Noise** | Audible | Silent |
| **Cost** | ~$0.02/GB | ~$0.10/GB |

## 3. The Storage Hierarchy

```
Registers    < 1 ns    ← Inside CPU
Cache         ~3 ns    ← Inside CPU
RAM           ~50 ns   ← DIMM slots
SSD         ~100 μs    ← SATA/NVMe
HDD          ~10 ms    ← SATA
```

Each level is ~10-100x slower than the level above it.

## 4. Summary

- HDDs are **slower but cheaper** per GB
- SSDs are **faster, quieter, more durable**
- Modern computers use SSDs for OS/apps, HDDs for bulk storage
