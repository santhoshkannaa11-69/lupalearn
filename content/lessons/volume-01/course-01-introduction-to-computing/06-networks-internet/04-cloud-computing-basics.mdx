## Cloud Computing Basics

**Cloud computing** is delivering computing services (servers, storage, databases, software) over the Internet.

### Cloud Service Models

| Model | What You Get | Example |
|-------|-------------|---------|
| **IaaS** | Virtual servers, storage | AWS EC2, Google Compute |
| **PaaS** | Platform to deploy apps | Heroku, Vercel, Railway |
| **SaaS** | Ready-to-use software | Gmail, Google Docs, Slack |

### Benefits of the Cloud

| Benefit | Description |
|---------|-------------|
| **Scale on demand** | Use more resources when needed |
| **Pay for what you use** | No upfront hardware costs |
| **Global reach** | Deploy servers worldwide |
| **Managed maintenance** | Provider handles hardware/security |
| **High availability** | Redundant systems prevent downtime |

### Major Cloud Providers

| Provider | Known For |
|----------|-----------|
| **Amazon Web Services (AWS)** | Largest market share, 200+ services |
| **Microsoft Azure** | Enterprise integration |
| **Google Cloud Platform (GCP)** | Data analytics, AI/ML |
| **Cloudflare** | CDN, security, edge computing |
