## 1. Hook

You're using the cloud right now. LupaLearn runs on Railway — a cloud platform. Your code stays on a server somewhere, and you access it through a browser. You never think about where the server is or who maintains it.

## 2. Cloud Service Models

| Model | You Manage | Provider Manages | Example |
|-------|-----------|-----------------|---------|
| **IaaS** | Apps, data, OS | Virtual servers, networking | AWS EC2 |
| **PaaS** | Apps only | Everything below | Railway, Vercel |
| **SaaS** | Nothing | Everything | Gmail, Google Docs |

## 3. Major Providers

| Provider | Known For |
|----------|-----------|
| **AWS** | Largest, 200+ services |
| **Azure** | Enterprise integration |
| **GCP** | Data analytics, AI |

## 4. Summary

- The cloud = **someone else's computers** you access over the Internet
- Three models: **IaaS**, **PaaS**, **SaaS** — decreasing control, decreasing responsibility
- Cloud computing **scales on demand** and **charges for what you use**
