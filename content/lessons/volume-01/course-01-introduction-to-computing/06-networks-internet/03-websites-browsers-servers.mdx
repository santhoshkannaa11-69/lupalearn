## 1. Hook

LupaLearn is currently running on a server somewhere, sending this text to your browser. Every word you read traveled across the Internet as data packets, was reassembled by your browser, and rendered as this page.

## 2. The Client-Server Model

```
You (Client)                  Server
┌─────────┐   Request →     ┌─────────┐
│ Browser │────────────────→│  Web    │
│         │←───────────────│  Server │
└─────────┘   Response      └─────────┘
```

## 3. What Happens When You Visit a Page

```
1. You type "lupalearn.app"
2. DNS → finds IP address
3. Browser sends HTTP request
4. Server processes request (may query DB)
5. Server sends back HTML + CSS + JS
6. Browser renders the page
```

## 4. Summary

- **Clients** (browsers) make requests
- **Servers** process requests and send responses
- "The Cloud" is just someone else's servers that you rent over the Internet
