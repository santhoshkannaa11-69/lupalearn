## What Is a Network?

A **computer network** is two or more computers connected to share resources and data.

### Network Types

| Type | Range | Example |
|------|-------|---------|
| **PAN** | Personal (1m) | Bluetooth headphones to phone |
| **LAN** | Local (building) | Office network, home Wi-Fi |
| **MAN** | Metropolitan (city) | City-wide Wi-Fi |
| **WAN** | Wide (world) | The Internet |

### Why Networks Matter

| Benefit | Description |
|---------|-------------|
| **File sharing** | Access files from any device |
| **Resource sharing** | Share printers, scanners |
| **Communication** | Email, chat, video calls |
| **Internet access** | Connect to the global network |

### Network Hardware

| Device | Function |
|--------|----------|
| **Router** | Connects networks together |
| **Switch** | Connects devices within a network |
| **Modem** | Connects to the internet service provider |
| **Access point** | Provides wireless connectivity |

Networking is the foundation of the Internet, cloud computing, and virtually all modern technology.
