## 1. Hook

Your phone connects to Wi-Fi, which connects to your ISP, which connects to a backbone, which routes through multiple cities, before reaching a server thousands of miles away — all in the time it takes for a web page to load.

## 2. Network Types

| Type | Range | Example |
|------|-------|---------|
| **PAN** | ~1m | Bluetooth headphones |
| **LAN** | Building | Office Wi-Fi |
| **WAN** | World | The Internet |

## 3. Network Hardware

| Device | Job |
|--------|-----|
| **Router** | Connects networks together |
| **Switch** | Connects devices within a network |
| **Modem** | Connects to ISP |
| **Access Point** | Wireless connectivity |

## 4. Summary

- A network connects computers to **share resources** and **communicate**
- Networks range from **personal (PAN) to global (WAN)**
- Routers, switches, and modems are the physical infrastructure
