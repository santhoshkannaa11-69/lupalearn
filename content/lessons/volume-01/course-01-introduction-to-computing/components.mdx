## Computer Components

Every computer has these essential components:

| Component | Function | Analogy |
|-----------|----------|---------|
| **CPU** | Executes instructions | The brain |
| **RAM** | Temporary memory while running | The desk space |
| **Storage** | Permanent data storage | The filing cabinet |
| **Motherboard** | Connects all components | The nervous system |
| **GPU** | Handles graphics rendering | The artist |
| **PSU** | Provides power | The heart |

### How They Work Together

1. **CPU** fetches instructions from **RAM**
2. **RAM** loads data from **Storage**
3. **CPU** processes the data
4. Results are sent to **output devices** (monitor, speakers)
5. **Storage** saves permanent changes

All components communicate through the **motherboard**, which acts as the central hub connecting everything.
