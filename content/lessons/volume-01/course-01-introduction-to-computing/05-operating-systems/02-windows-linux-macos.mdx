## Windows, Linux & macOS

Three major operating systems dominate computing.

### Windows (Microsoft)

| Aspect | Detail |
|--------|--------|
| **Market share** | ~75% of desktops |
| **UI** | Graphical, start menu, taskbar |
| **Software** | Largest selection of games and business apps |
| **File system** | NTFS |
| **Best for** | Gaming, enterprise, general use |

### macOS (Apple)

| Aspect | Detail |
|--------|--------|
| **Market share** | ~15% of desktops |
| **UI** | Clean, integrated, gesture-based |
| **Software** | Strong creative tools (Final Cut, Logic) |
| **File system** | APFS |
| **Best for** | Creative work, development, Apple ecosystem |

### Linux (Open Source)

| Aspect | Detail |
|--------|--------|
| **Market share** | ~5% of desktops, 90%+ of servers |
| **UI** | Many options (GNOME, KDE, etc.) |
| **Software** | Free, open source, developer tools |
| **File system** | ext4, Btrfs, ZFS |
| **Best for** | Servers, development, embedded systems |

### Which Should You Choose?

| You want to... | Use |
|----------------|-----|
| Play games | Windows |
| Build websites | Any (but macOS is popular) |
| Run servers | Linux |
| Make music/video | macOS |
| Learn programming | Linux or macOS |
| General use | Windows or macOS |

All three are used by developers. The best choice depends on your goals.
