## 1. Hook

Linux runs 90% of the cloud, macOS dominates creative work, and Windows covers the rest of the desktop market. Three operating systems, three philosophies — each excelling in different areas.

## 2. Comparison

| Aspect | Windows | macOS | Linux |
|--------|---------|-------|-------|
| **Owner** | Microsoft | Apple | Community |
| **Market** | ~75% desktop | ~15% desktop | ~90%+ cloud |
| **Price** | Hardware + license | Hardware included | Free |
| **Philosophy** | Compatible | Integrated | Open |
| **Best for** | Gaming, business | Creative, dev | Servers, dev |

## 3. Which Should You Use?

| You want to... | Recommended OS |
|----------------|---------------|
| Play games | Windows |
| Build websites/startups | macOS or Linux |
| Run servers | Linux |
| Make music/video | macOS |
| Learn programming | Any (all work) |

## 4. Summary

- **Windows** dominates gaming and enterprise
- **macOS** excels in creative work and developer experience
- **Linux** powers the cloud and is free/open-source
- For learning programming, **all three** are fine choices
