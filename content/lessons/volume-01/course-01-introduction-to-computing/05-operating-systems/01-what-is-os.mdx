## 1. Hook

When you open three applications at once, how does your laptop decide which one gets the CPU's attention? How do they share memory without interfering with each other? The answer is the most important software you never think about.

## 2. What the OS Does

| Function | Description | Everyday Example |
|----------|-------------|------------------|
| **Process management** | Runs and schedules programs | Switching between apps |
| **Memory management** | Allocates RAM | Keeping 10 tabs open |
| **File system** | Organizes data | The folder structure |
| **Device management** | Controls hardware via drivers | Printing a document |
| **Security** | Controls access | User accounts, permissions |

## 3. Mental Model

The OS is like a **building manager**:

- Allocates space (memory) to different tenants (applications)
- Ensures everyone gets time on shared facilities (CPU)
- Handles maintenance (device drivers)
- Enforces security (door locks)
- Manages shared resources (heating, water)

## 4. Summary

- The OS is the **foundation software** that everything else runs on top of
- It manages: **processes, memory, files, devices, security**
- Without an OS, every program would need to handle these itself
