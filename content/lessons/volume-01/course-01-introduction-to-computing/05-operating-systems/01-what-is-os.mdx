## What Is an Operating System?

An **Operating System (OS)** is the most important software on a computer. It manages all hardware and software and provides services for programs.

### What the OS Does

| Function | Description |
|----------|-------------|
| **Process management** | Runs and schedules programs |
| **Memory management** | Allocates RAM between programs |
| **File system** | Organizes data on storage |
| **Device management** | Communicates with hardware via drivers |
| **Security** | Controls access and permissions |
| **User interface** | Provides a way to interact with the computer |

### Without an OS

Without an OS, every program would need to:

- Know exactly what hardware was installed
- Include drivers for every device
- Manage its own memory allocation
- Handle file storage directly

This would make software development impossibly complex.

### The OS as a Layer

```
Hardware
    ↓
Operating System
    ↓
Applications
    ↓
User
```

The OS abstracts away hardware complexity so developers can focus on features.
