## 1. Hook

Every file on your computer lives somewhere in a tree of folders. This structure — first developed in the 1960s — is so fundamental that almost every operating system uses the same basic design.

## 2. File System Structure

```
/ (root)
├── home/
│   └── user/
│       ├── Documents/
│       ├── Downloads/
│       └── Projects/
├── etc/          (config files)
├── usr/          (program files)
└── var/          (logs, data)
```

## 3. File Systems Compared

| System | Used By | Max File Size |
|--------|---------|---------------|
| **NTFS** | Windows | 16 EB |
| **APFS** | macOS | 8 EB |
| **ext4** | Linux | 16 TB |
| **FAT32** | USB drives | 4 GB |

## 4. Summary

- The file system organizes data into a **tree of directories**
- Different OSs use different file systems (NTFS, APFS, ext4)
- Understanding the hierarchy helps you navigate any computer
