## Graph Theory

A **graph** is a collection of **vertices** (nodes) connected by **edges**.

### Graph Terminology

| Term | Definition |
|------|------------|
| Vertex/Node | A point in the graph |
| Edge | A connection between two vertices |
| Path | A sequence of edges between vertices |
| Cycle | A path that starts and ends at the same vertex |
| Tree | A graph with no cycles |

### Types of Graphs

- **Directed** — edges have direction (like Twitter follows)
- **Undirected** — edges are bidirectional (like Facebook friends)
- **Weighted** — edges have costs or distances
- **Cyclic** — contains cycles
- **Acyclic** — no cycles (tree)

### Applications

- Social networks (friend connections)
- Maps and navigation (shortest path)
- Web pages (hyperlink graph)
- Computer networks (routing)
- LupaLearn's knowledge graph!
