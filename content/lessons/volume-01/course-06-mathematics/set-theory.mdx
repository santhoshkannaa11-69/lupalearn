## Set Theory

A **set** is an unordered collection of distinct objects.

### Set Notation

```
A = {1, 2, 3, 4, 5}
B = {3, 4, 5, 6, 7}
```

### Set Operations

| Operation | Symbol | Meaning | Example |
|-----------|--------|---------|---------|
| Union | A ∪ B | All elements in either | {1,2,3,4,5,6,7} |
| Intersection | A ∩ B | Elements in both | {3,4,5} |
| Difference | A − B | Elements in A not in B | {1,2} |

### In Programming

Sets are fundamental data structures. Python has `set`, JavaScript has `Set`, and SQL uses set operations (UNION, INTERSECT, EXCEPT).
