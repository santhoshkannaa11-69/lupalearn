## Command Line Basics

The **command line** (terminal) is a text-based interface to your computer.

### Essential Commands

| Command | What It Does |
|---------|-------------|
| `ls` | List files and directories |
| `cd` | Change directory |
| `pwd` | Print working directory |
| `mkdir` | Create a new directory |
| `touch` | Create an empty file |
| `cp` | Copy files |
| `mv` | Move/rename files |
| `rm` | Delete files |
| `cat` | Show file contents |

### Why Learn the Command Line?

- Faster than GUI for many tasks
- Essential for development (Git, Node.js, Docker)
- Scriptable — automate repetitive tasks
- Runs on every server you'll ever use

### Try It

Open the **Playground** terminal and try:
```bash
ls -la
mkdir my-project
cd my-project
touch index.js
```
