## Open Source Software

**Open source software** (OSS) is software with source code that anyone can inspect, modify, and distribute.

### Common Licenses

| License | Freedom | Requirements |
|---------|---------|-------------|
| **MIT** | Maximum | Keep the license notice |
| **Apache 2.0** | Maximum | Keep notice + state changes |
| **GPL** | High | Must also be open source |
| **BSD** | Maximum | Keep notice |

### Why Open Source Matters

- **Transparency** — anyone can audit the code
- **Community** — millions of contributors worldwide
- **Quality** — more eyes find more bugs
- **Free** — no licensing costs

### How to Contribute

1. Find a project you use
2. Read CONTRIBUTING.md
3. Start with documentation or small bug fixes
4. Submit a Pull Request (PR)
