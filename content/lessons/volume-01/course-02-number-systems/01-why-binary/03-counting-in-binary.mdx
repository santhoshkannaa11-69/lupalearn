## 1. Hook

In decimal, after 9 comes 10. In binary, after 1 comes 10. Same symbol, totally different meaning. Let's learn how to count like a computer.

## 2. Core Concept

### Counting 0 to 7 in Binary

| Decimal | Binary | Why? |
|---------|--------|------|
| 0 | 000 | |
| 1 | 001 | 1 |
| 2 | 010 | 2 |
| 3 | 011 | 2 + 1 |
| 4 | 100 | 4 |
| 5 | 101 | 4 + 1 |
| 6 | 110 | 4 + 2 |
| 7 | 111 | 4 + 2 + 1 |

### Powers of 2 (Memorize These)

```
2⁰ = 1     2⁴ = 16
2¹ = 2     2⁵ = 32
2² = 4     2⁶ = 64
2³ = 8     2⁷ = 128
```

## 3. Mental Model — Pattern Recognition

Binary digits follow a predictable rhythm when you count up:

```
000 → 001 → 010 → 011 → 100 → 101 → 110 → 111
                                  ↑
                     The rightmost toggles every step
                            Second toggles every 2 steps
                           Third toggles every 4 steps
```

## 7. Summary

- Binary counting follows the same positional rules as decimal
- Each binary position represents a **power of 2**
- The rightmost bit toggles every step, the next every 2 steps, etc.
- Memorize powers of 2: 1, 2, 4, 8, 16, 32, 64, 128
