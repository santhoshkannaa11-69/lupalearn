## 1. Hook

One bit can only store a 0 or 1 — not useful on its own. But group 8 bits together, and you can store 256 different values — enough for every letter, number, and symbol on your keyboard.

## 2. Core Concept

A **byte** is 8 bits. It's the standard unit for measuring data.

### Storage Units

| Unit | Size | Rough Equivalent |
|------|------|------------------|
| **1 byte** | 8 bits | One character |
| **1 KB** | 1,024 bytes | A short paragraph |
| **1 MB** | 1,024 KB | A photograph |
| **1 GB** | 1,024 MB | A full-length movie |
| **1 TB** | 1,024 GB | A hard drive |

### Why 1,024?

Computers use binary, so storage units are based on powers of 2:

```
1 KB = 2¹⁰ = 1,024 bytes (not 1,000)
1 MB = 2²⁰ = 1,048,576 bytes
1 GB = 2³⁰ = 1,073,741,824 bytes
```

## 4. Summary

- **1 byte = 8 bits**
- Storage units: **KB, MB, GB, TB** — each is 1,024× the previous
- Binary-based (1,024) vs decimal-based (1,000) marketing causes confusion
