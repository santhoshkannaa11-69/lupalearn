## 1. Hook

You've learned to convert decimal to binary. Reversing the process is even simpler — just add up the place values where there's a 1.

## 2. Core Concept

### The Positional Value Method

```
Binary:  1  1  0  1  0  1  0  0
Places: 128 64 32 16  8  4  2  1
Add:   128+64 +0 +16 +0 +4 +0 +0 = 212
```

### Step-by-Step

```
Position 7 (128): 1 × 128 = 128
Position 6  (64): 1 × 64  =  64
Position 5  (32): 0 × 32  =   0
Position 4  (16): 1 × 16  =  16
Position 3   (8): 0 × 8   =   0
Position 2   (4): 1 × 4   =   4
Position 1   (2): 0 × 2   =   0
Position 0   (1): 0 × 1   =   0
                       Total: 212
```

## 4. Quick Conversion Table

|--------|---------|
| 00000000 | 0 |

| 10000000 | 128 |
| 11111111 | 255 |

## 5. Summary

- Binary → decimal: **add the place values** where bits are 1
- Each 1 contributes **2^(position)** to the total
- The leftmost bit (MSB) has the highest value
