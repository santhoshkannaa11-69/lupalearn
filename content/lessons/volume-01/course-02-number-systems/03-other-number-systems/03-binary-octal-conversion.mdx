## 1. Hook

Converting between binary and octal is the simplest of all base conversions — group binary digits into sets of 3 and convert each group.

## 2. Core Concept

### Binary → Octal

Group binary digits into sets of 3, starting from the right:

```
Binary: 101 110 011
Octal:   5   6   3

Result: 563₈
```

### Octal → Binary

Convert each octal digit to 3 binary digits:

```
Octal:   7   5   1
Binary: 111 101 001

Result: 111101001₂
```

## 4. Summary

- **Group by 3** for binary ↔ octal conversion
- Add leading zeros to the leftmost group if needed
- Each octal digit maps to exactly 3 binary digits
