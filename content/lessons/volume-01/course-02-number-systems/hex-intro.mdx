## Hexadecimal Numbers

**Hexadecimal** (base-16) uses 16 digits: 0-9 and A-F. It's used extensively in computing because it represents binary more compactly.

### Hex Digits

| Decimal | Binary | Hex |
|---------|--------|-----|
| 0 | 0000 | 0 |
| 1 | 0001 | 1 |
| 10 | 1010 | A |
| 15 | 1111 | F |

### Binary to Hex

Group binary digits into sets of 4 (from right):
```
Binary: 1101 1011
Hex:    D    B
→ 0xDB
```

### Common Uses

- **Memory addresses** — displayed in hex (`0x7FFF`)
- **Color codes** — `#FF0000` is red (RGB in hex)
- **MAC addresses** — `00:1A:2B:3C:4D:5E`
