## 1. Hook

Binary division uses the same long-division process you learned in school — just with only 0s and 1s.

## 2. Core Process

```
         1101  (13 ÷ 3 = 4 remainder 1)
       ─────
011 │ 001111
       011
       ───
        011
        011
        ───
         0011
           11
          ───
           0
```

## 5. Summary

- Binary division uses **the same long-division algorithm** as decimal
- Each step: does the divisor fit? 1 if yes, 0 if no
- Much simpler than decimal because each digit is either 0 or 1
