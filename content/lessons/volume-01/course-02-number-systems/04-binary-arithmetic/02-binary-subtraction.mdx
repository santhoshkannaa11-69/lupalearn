## 1. Hook

Subtraction in binary is like decimal subtraction — when you need to subtract 1 from 0, you borrow from the next column. But in binary, borrowing is simpler.

## 2. Core Rules

```
0 - 0 = 0
1 - 0 = 1
1 - 1 = 0
0 - 1 = 1  (borrow 1 from the left)
```

## 4. Summary

- Binary subtraction follows the same process as decimal
- **0 - 1 requires borrowing** from the next position
- Borrowing propagates left until a 1 is found
