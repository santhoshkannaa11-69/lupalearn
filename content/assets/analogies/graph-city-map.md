# City Map Analogy for Graphs

## The Analogy
A city map is a graph.
- **Cities** = vertices
- **Roads** = edges
- **One-way streets** = directed edges
- **Distance** = edge weight
- **Shortest route** = shortest path problem

## Visual
```
🏠 City A —— 🏠 City B
     \          /
      🏠 City C
     /          \
🏠 City D —— 🏠 City E
```

## Key Insights
| Graph Concept | Map Concept |
|--------------|-------------|
| Vertex | A city |
| Edge | A road |
| Directed edge | One-way street |
| Undirected edge | Two-way road |
| Degree | Number of roads at a city |
| Path | Route between cities |
| Cycle | Round trip |
