# ADR-002: Three Learning Modes

## Status

Accepted

## Context

Different learners need different entry points. Beginners want structure. Experienced developers want fast access to specific topics. Goal-oriented learners want personalized paths.

## Decision

Provide three modes sharing the same content:
- **Guided** — prerequisite-locked curriculum progression
- **Explorer** — search-first, no locks, all content accessible
- **AI** — personalized roadmap generated from user goals

## Consequences

- Three experiences from one content base
- Users can switch modes at any time
- Onboarding asks "what do you want to do?" and selects the mode
